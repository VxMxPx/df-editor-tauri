# Keyboard Shortcut Display

Add a reusable keyboard-shortcut display component.

## Requirements

- Add `src/features/ui/kbd.ui.svelte`.
- Accept a required `shortcut: string[]` prop.
- Render each shortcut part in a semantic `<kbd>` element, separated by `+`.
- Export `Kbd` from `@df/ui`.
- Do not normalize shortcut names or add keyboard behavior.

## Acceptance

- `<Kbd shortcut={["CMD", "W"]} />` renders `CMD + W`.
